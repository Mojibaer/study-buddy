Algorithmen und Datenstrukturen

Ein Algorithmus ist eine eindeutige Handlungsvorschrift zur Lösung eines Problems.

Wichtige Sortieralgorithmen:
- Bubble Sort - O(n²) Zeitkomplexität, einfach aber ineffizient
- Quick Sort - O(n log n) im Durchschnitt, sehr effizient
- Merge Sort - O(n log n) garantiert, stabil
- Heap Sort - O(n log n), in-place Sortierung

Datenstrukturen:
- Arrays - Direkter Zugriff, feste Größe
- Linked Lists - Dynamische Größe, sequentieller Zugriff
- Stacks - LIFO (Last In First Out)
- Queues - FIFO (First In First Out)
- Trees - Hierarchische Struktur
- Hash Tables - O(1) Zugriff im Durchschnitt

Rekursion ist ein wichtiges Konzept, bei dem eine Funktion sich selbst aufruft. 
Die Fibonacci-Folge ist ein klassisches Beispiel für Rekursion.

Big-O-Notation beschreibt die Zeitkomplexität von Algorithmen.
